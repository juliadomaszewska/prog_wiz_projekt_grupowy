# MemoryGame

This folder contains a separate Avalonia-based Memory card game for the group project.

- `MemoryGame.csproj` is the standalone Avalonia app.
- `Views/` contains the main game window.
- `ViewModels/` contains the game logic and card state.

Use `dotnet run --project MemoryGame/MemoryGame.csproj` to launch the game.
